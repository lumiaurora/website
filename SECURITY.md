# Security Policy

## Supported versions

Security fixes are applied to the latest version on `main`.

## Reporting a vulnerability

Please do not open public issues for security-sensitive problems.

Report security concerns by emailing:

- `yukicodes@outlook.com`

Include:

- a clear description of the issue
- affected files or behavior
- reproduction steps, if possible
- any suggested remediation

I will review reports as quickly as possible and coordinate a fix before public disclosure when needed.
