# Contributing

Thanks for helping improve this portfolio template.

## Before you start

- Check existing issues and pull requests first.
- Keep changes focused and easy to review.
- If you are changing content examples, preserve the overall template quality.

## Local setup

```bash
npm install
npm run build
npm run check
```

For a local preview:

```bash
npm run dev
```

## Editing rules

- Update the JSON files in `content/` for copy changes.
- Update `src/template.html` only when the page structure needs to change.
- Do not hand-edit `index.html`; regenerate it with `npm run build`.
- Keep CSS and JS changes small and well explained.

## Pull request checklist

- Run `npm run check`.
- Include regenerated `index.html` if the template or content changed.
- Explain what changed and why.
- Add screenshots when the UI changes visibly.

## Good first contributions

- documentation improvements
- accessibility fixes
- responsive polish
- clearer customization instructions
- additional validation checks
