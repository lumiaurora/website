# Code of Conduct

## Our standard

This project aims to be welcoming, respectful, and useful for contributors of all experience levels.

Examples of positive behavior:

- being respectful and constructive
- assuming good intent
- giving actionable feedback
- helping others learn

Examples of unacceptable behavior:

- harassment or discrimination
- personal attacks
- bad-faith disruption
- sharing private information without consent

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact:

- `yukicodes@outlook.com`
